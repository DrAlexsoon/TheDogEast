**全局定义累计基数**
```java
class Solution {
    int total = 0;
    public TreeNode convertBST(TreeNode root) {
        if (root != null) {
            convertBST(root.right);
            total += root.val;
            root.val = total;
            convertBST(root.left);
        }
        return root;
    }
}
```
